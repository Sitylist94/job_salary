# Job Salary Prediction

Ce projet utilise le machine learning pour prédire les salaires à partir de caractéristiques comme le poste, l'expérience, les compétences, et l'éducation.

## Contenu du projet

- `src/` : Code source du projet.
  - `main.py` : Script principal pour entraîner et tester le modèle.
  - `figures/` : Dossier pour stocker les figures et graphiques générés (vide au départ avec `.gitkeep`).
- `data/` : Contient le dataset (non inclus dans le repo pour ne pas partager les données sensibles).
- `.gitignore` : Pour ignorer les fichiers temporaires et le dossier `data/`.

## Installation

1. Cloner le projet :

```bash
git clone https://github.com/Sitylist94/job_salary.git
cd job_salary
```

2. Créer et activer un environnement virtuel :
```bash
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate # Linux / Mac
```

3. Installer les dépendances :
```bash
pip install -r requirements.txt
```
